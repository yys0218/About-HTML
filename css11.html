About HTML

가상 선택자

: 실제 html의 요소에 직접적으로 일치하지 않는, 요소의 특정 상태나 문서 구조를 기반으로 적용되는 선택자

가상 요소
: 실제로는 존재하지 않는 문서의 부분을 생성하여 스타일을 적용할 수 있도록 함.
- 최신 css에서는 ::을 사용하는 것이 좋다고 함( :으로 해도 작동되긴함 )

before/after 가상 요소

: 가상의 content(내용)를 before(이전), after(이후)에 추가할 수 있는 선택자
- content 속성에 원하는 문자열을 지정하여 사용
태그:before { content:"태그 전에 추가할 내용"; 스타일규칙 지정 가능; }
h4:before{ content:"문단 시작" color:gray; }

태그:after { content:"태그 뒤에 추가할 내용"; 스타일규칙 지정 가능; }
h4:after{ content:"문단 끝" color:gray; }

*선택자 지정 시 세미콜론 뒤에 공백이 있으면 안됨!

< h4 > p:before ----p:after < /h4 >

first-line 가상 요소

: 요소의 첫 번째 줄이 되는 내용에 스타일 적용
- 태그 작성 기준으로 고정된 영역이 아닌, 브라우저 창의 크기에 반응하는 '첫째 줄'에 유동적으로 적용됨
태그:first-line { 스타일 규칙; }
.line p:first-line { color:red; font-weight:bold; }
< div class="line" >

< p >클래스가 line인 div태그의 p태그 시작--------------------------------------브라우저 창의 크기를 바꿔가면서 스타일이 적용되는 범위를 확인해보세요. ----------------------------------------태그 종료 < /p >

< /div >
*가상 요소로 지정한 스타일 - 하나의 태그에 중복 적용 가능 / ,로 여러 가상요소에 하나의 스타일 지정 가능

first-letter 가상 요소

: 요소의 첫 번째 글자를 선택하여 스타일 적용
태그:first-letter { 스타일 규칙; }
.letter p:first-letter { color:red; font-weight:bold; font-size:3em }
< div class="letter" >
< p >

클래스가 letter인 div의 p태그의 첫글자에 스타일을 적용함

< /p >
< /div >
