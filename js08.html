스코프

scope: 해당 변수를 사용할 수 있는 범위

- scope에 따라,< script >내 에서 사용 가능한 전역 변수와
함수 또는 블럭 내에서 사용 가능한 지역 변수로 구분

전역 변수

: < script >내, 함수 외부에서 선언된 변수
- 모든 스크립트(전체 코드) 내에서 사용 가능

지역 변수

: 함수 또는 블럭{} 내부에서 선언된 변수
- 변수가 선언된 함수, {} 외부에서 접근 불가

< script > function addNum(){
   var sum = 10+20; <= 함수 내 선언&초기화된 지역 변수
   console.log(sum); <= 함수 내 사용 가능
}
addNum(); *함수 정의 후 호출(사용)해줘야함.
//console.log(sum)  <= 함수 밖에서 변수 사용 불가(지역 변수에 접근 불가, sum is undefined)
</script>

*키워드 없이 지역 변수 선언시,
암묵적으로, 전역 변수가 되어 '전역 오염'이 됨
>>변수 선언시 꼭 키워드를 붙이자!

변수 선언시 사용하는 키워드: var, let, const

< script > function addNum2(){
   plus = 30+40; <= 함수 내 키워드 없이 선언된 변수
   console.log(plus); <= 함수 내 사용 가능
}
addNum();
console.log(plus)  <= 함수 밖에서도 사용됨
</script>
