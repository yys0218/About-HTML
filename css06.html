클래스 선택자

Class Selector
: 특정 "클래스"로 지정한 요소에만 스타일을 적용할 때
- 클래스는 문서 내 여러 개의 요소에 지정될 수 있음

클래스 선택자 사용
- 스타일을 적용하고자 하는 클래스명앞에 .(마침표)를 붙여 클래스 선택자를 지정
.accent {
      border:1px solid #000;
      padding:5px;
}
- 요소명.클래스명 으로 사용시 해당 요소의 클래스에만 스타일이 적용됨
p.bg {
      background-color:#skyblue;
}

클래스 지정
- 스타일을 적용하고싶은 태그 내 class="클래스명" 지정
*클래스명으로 다른 선택자와 구별, 이름은 임의로 지정 가능
< h3 class="accent" > accent 클래스가 지정된 h3태그 < /h3 >

accent 클래스가 지정된 h3태그

- 하나의 요소에 2개 이상의 클래스 지정 가능
>> 공백으로 구분 class="이름 이름"
< p class="accent bg" > accent, bg 클래스가 지정된 p태그

accent, bg 클래스가 지정된 p태그

< h3 class="bg" > ... < /h3 >

bg클래스가 지정된 h3태그

아이디 선택자

ID Selector
: 웹 문서에서 특정 "id"로 지정한 요소에만 스타일을 적용할 때 사용
- 하나의 id는 문서 내 하나의 요소에만 적용될 수 있음

id선택자 사용
- 스타일을 적용하고자 하는 id앞에 #을 붙여 id선택자를 지정
#container {
      width:500px;
      margin:10px auto;
      padding:10px;
      border:1px solid #000;
}
id 지정
- 스타일을 적용하고 싶은 태그 내 id="아이디명" 지정. 이름은 임의로 지정
< div id="container" >...< /div >


class / id 선택자 차이

class 선택자: 문서에서 여러 번 적용 가능

id 선택자: 문서에서 한 번만 적용
