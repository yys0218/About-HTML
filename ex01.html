About HTML

HTML(Hyper Text Markup Langauge)

Hyper Text:'서로 연결된 텍스트'
>>웹 페이지에서 링크를 통해 다른 문서로 이동할 수 있는 기능

Markup Language
: 데이터 구조화하고 웹 페이지의 콘텐츠를 표시하기 위해 사용하는 언어
태그< >를 사용하여 문서의 구조를 정의

>>HTML은 웹 페이지의 기본 뼈대
CSS, JavaScript와 함께 웹 개발에 사용

태그
: < >로 감싸져 있음 - <..>시작 태그 종료태그가 있다.
- 싱글 태그: 종료 태그가 없이 하나만 사용되는 태그
< !-- HTML에서 주석 넣기 -->
- 프로그램 수행과 관련없이 참고사항을 넣을 수 있음
- 브라우저를 통해 직접 볼 수 없고 아무런 영향을 주지 않음
- 소스 파일을 볼 때 확인 가능
