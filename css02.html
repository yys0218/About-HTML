인라인 스타일

: 태그 내에 직접 style 속성을 사용
- 해당 태그에만 스타일을 적용
< p style="color:blue" >

이 태그에만 파랑색 글씨 적용

< /p >

내부 스타일 시트

Internal StyleSheets
:문서(html)에 사용할 스타일을 같은 문서 내에 정의하는 방식
- 이 때의 모든 스타일 정보는 < head >< /head > 태그 안의
< style > < /style > 태그 안에서 작성되어야함
<head> <style> h3 {
   padding:10px;
   background-color:skyblue;
   color:#fff;
}
</style> </head>

내부 스타일 시트가 적용된 h3태그

외부 스타일 시트

External StyleSheet
:일관성 있는 스타일을 사용하기 위해 css코드를 별도 파일로 저장, 필요할 때 가져오는 방식
- 별도 파일은 .css 확장자로 저장됨
ㄴ 별도 파일 내에서는 < style >내 작성할 필요X
style.css 생성 h4{
   padding: 10px;
   background-color:greenyellow;
   color:magenta;
}

- 외부 스타일 시트를 연결하는 싱글태그< link >는 < head > 내에 존재해야함
<head> <style> < link rel="stylesheet" href="외부 스타일 시트의 파일 경로" >
</style> </head> * rel=stylesheet는 고정값

외부 스타일 시트가 적용된 h4태그


src : 이미지, 스크립트, iframe 등 외부 리소스 삽입용 - 브라우저는 src리소스를 즉시 다운/처리함
    < img src="image의 주소" > < script src="script.js" >

href : 하이퍼링크 또는 스타일 시트의 경로 지정 - 브라우저는 href를 즉시 다운하지 않고, 필요할 때 접근할 수 있도록 처리함
      < a href="https://example.com" >  < link href="style.css" rel="stylesheet" >
