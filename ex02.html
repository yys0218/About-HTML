About HTML

< head >

< head > : 제목을 포함하는 HTML 문서의 머리(head)부분
< title > 웹페이지 제목 < /title >
- 브라우저 창에 보이지 않음(탭의 이름)
- 본문과 구별되어, body를 설명하는 태그
< /head>

< body >

< body >
- HTML 문서의 본체(body)
- 문서의 내용, 구성이 들어가는 주된 부분 (문서 제목 등도 모두)
< /body >

< br >

줄바꿈 태그< br >
이 있는 부분에서 줄바꿈이 일어남

< h1 >

머릿글자 (headings): 단계별로 제목을 표시
머릿글자는 일반 문서의 글자에 비해 크고 더 굵은 글자로 표현됨
앞 문장과 뒷 문장을 분리하기 위해 줄바꿈이 적용되고, 위 아래 공백이 생김
시작 태그와 종료 태그 사이에 내용을 넣어 사용

< h1 >가장 큰 제목1< /h1 >

< h2 >제목2< /h2 >

< h3 >제목3< /h3 >

< h4 >제목4< /h4 >

< h5 >제목5< /h5 >
< h1 >제목6< /h6 >
- 일반적으로 HTML 문서는 대소문자를 구별하지 않음
