About HTML

< form >

- HTML 문서에서 데이터를 주고 받을 때 사용하는 태그
- 데이터를 입력받을 수 있는 태그(요소)를 하위 요소로 가짐
< form >
  < 내부태그 속성="값" >< /내부 태그 >
< /form >
- 요소(element)들은 속성의 이름(name)=값(value)이 짝을 이룸

< form >
  < input type="text" name="st_name" value="홍길동"... >
  < button name="fruit" value="apple" ... >사과< /button >
< /form >

* 요소의 이름은 자바로 치면, 변수명인 셈 >> 각 요소마다 달라야함.
* 값이 주어진 요소를 '파라미터'라고 함.

form태그의 속성
method="get|post": 폼 데이터가 서버로 제출될 때 사용되는 HTTP메소드를 지정
action="URL": 폼 데이터를 서버로 보낼 때 데이터가 도착할 URL명시

form태그의 하위 요소

< input >
:사용자로부터 입력을 받을 수 있는 입력 필드가 생성됨
- 텍스트나 이미지와 같은 콘텐츠를 삽입할 수 없음
- 싱글 태그

input 태그의 속성 - 이름과 값
name="이름" : input태그의 이름
value="값" : input태그에 부여된 초기값

input 태그의 속성 - type= ~
="text" : 단순 텍스트를 입력받음
ㄴ size="글자수" : 텍스트 입력창의 크기(한 번에 보여지는 글자수)
ㄴ maxlength="최대글자수" : 입력가능한 최대 글자 수를 지정
ㄴ placeholder="텍스트" : 입력창에 짧은 도움말을 명시함, 입력이 시작되면 사라짐

="password" : 입력되는 값이 가려져서 보임
="checkbox" : 같은 < form >내에서 중복 선택이 가능한 체크박스를 생성, 선택시 value의 값이 서버로 제출됨
="radio" : 같은 < form >내에서 중복 선택이 불가한 선택항목을 생성, 선택시 value의 값이 서버로 제출됨
선택항목들의 name은 같고, value는 모두 달라야 함!

="button" : value의 값(텍스트)를 가진 버튼 생성
="img" : 이미지를 통해 이벤트를 작동시킴
="hidden" : 이 속성이 사용된 태그는 브라우저에서 보이지 않음
