About HTML

< table >

: 표를 만들 때 사용
- < table >내 요소 3가지로 표가 만들어짐
ㄴ < tr > :표의 한 행(가로줄) Table Rows
  ㄴ < td > : 표의 칸 Table Data
   ㄴ < th > : 표의 제목 (표 맨 위에서 항목을 분류) Table Head(Title)

table에서 사용 가능한 속성
bgcolor="색상값" : 표의 바탕색 / 색상명 혹은 값
width="표 전체 가로 길이" : 표의 너비 / 픽셀 혹은 백분율로 지정
height="표 전체 세로 길이" : 표의 높이 / 픽셀 혹은 백분율로 지정
border="두께" : 표의 테두리 두께 / 픽셀로 지정
ㄴ미지정시 기본적으로, 테두리는 안보이는 상태
ㄴborder-collapse:collapse : 기본 2중으로 보여지는 테두리를 실선으로 바꿈 *기본값:separate
cellspacing="간격" : 표의 칸 사이의 간격 / 픽셀로 지정
cellpadding="간격" : 칸 안에서 간격 / 픽셀로 지정

< tr >
: 표의 한 행이 되는 태그
-table 내에서 반드시 한 쌍< tr >< /tr >이상 존재해야함

tr 태그에서 사용 가능한 속성
bgcolor="색상": 표 한 행의 바탕색 / 색상명 혹은 값
width="가로길이": 한 행의 폭을 지정
height="세로길이": 한 행의 높이를 지정
align=left / center / right
:한 행에서 내용의 수평 정렬
valign=top / middle / bottom
:한 행에서 내용의 수직 정렬

< td > , < th >
-표의 제목(th) 요소도 표의 칸(td)의 한 종류임.
-< td >가 1개도 들어있지 않은 < tr >은 표로 만들어지지 않음.
-< th >는 가운데정렬이 기본값임

td태그가 사용 가능한 속성
bgcolor="색상" : 칸의 바탕색
background="url" : 칸의 배경 이미지
width="가로" : 칸의 폭을 지정
height="세로" : 칸의 높이 지정
align=left / center / right
: 칸 내용의 수평 정렬 지정 - 디폴트 left
valign=top / middle / bottom
: 칸 내용의 수직 정렬 지정 - 디폴트 middle
colspan="가로로 합칠 칸의 갯수" : 해당 칸에서부터 가로 통합할 칸의 갯수 지정
rowspan="세로로 합칠 칸의 갯수" : 해당 칸에서부터 세로 통합할 칸의 갯수 지정
ㄴ오른쪽으로만 통합됨! 칸 개수 잘 볼것

< tr >
< th > 1번 행 1번 칸 th < /th > < th > 1번 행 2번 칸 th < /th >
< td > 2번 행 1번 칸 td < /td > < td > 2번 행 1번 칸 th < /td >
< /tr >
< /tr >

< /table >

>>모든 속성은 브라우저에 따라 표현의 차이가 있다
