About HTML

글자들을 표현하는 태그들

라인 모드(inline mode)와 블럭모드(block mode)
- 라인 모드: 자동적으로 줄바꿈을 하지 않음.
- 블럭 모드: 자동적으로 줄바꿈이 적용됨, 여러 줄의 문장 표현에 적합

라인 모드

- < b >, < strong >
:태그 안의 글자를 굵게 강조 표시
문장 중에< b >b태그로 묶어 굵게 표시되는 내용< /b >이고,
< strong >strong으로 강조 표시되는 내용< /strong >입니다.

- < i >, < cite >, < dfn >, < em >
:태그 안의 글자를 기울어진 글자체인 '이태릭'체로 표현한다.
< i >i 태그로 표시되는 내용< /i >
< cite >cite 태그로 표시되는 내용< /cite >
< dfn >dfn 태그로 표시되는 내용< /dfn >
< em >em 글자로 표시되는 내용< /em >

- < del >, < s >, < strike >
:태그 안의 글자를 지운 글자(취소선)로 표시
< del >del은 내용이 새롭게 업데이트된 상황에서 삭제된 내용을 나타냄 < /del >
< s >s는 잘못된(틀린) 텍스트를 강조하는 것. < /s >
< strike >줄 친 글자로 표시되는 내용< /strike >

- < ins >, < u >
:밑줄
< u >u는 단순히 내용에 밑줄을 칠 때 사용< /u >
< ins >ins는 내용이 업데이트된 상황에서 새롭게 추가된 내용을 나타냄 < /ins >

블럭 모드: 사용시 줄바꿈 자동 적용

- < p >
:태그 안의 문장을 하나의 문단으로 표시
시작과 끝(p태그의 위/아래)에 빈 줄을 삽입하면서 자동 줄바꿈 적용

< p > p태그 안에는 하나의 문장, 짧은 문단이 들어갑니다. < /p >


- < div >
:태그 안의 문장을 하나의 문단으로 표시
시작과 끝에 빈 줄을 삽입하지 않고, 자동 줄바꿈 적용
< div > div태그는 보통 레이아웃(스타일)적용을 위한 컨테이너로 쓰이며, 특별한 용도는 없다. < /div >

-address태그
:태그 안의 문장을 이태릭 글자로 된 하나의 문단으로 표시
시작과 끝에 빈 줄을 삽입하지 않고, 자동 줄바꿈 적용
< address >
address 태그는 내용의 저자, 조직 등에 대한 연락처 정보를 나타냅니다.
< /address >

- < pre >
:태그 안의 내용을 빈 칸, 줄바꿈, 탭(tab)까지 그대로 표현된 하나의 문단으로 표시
시작과 끝에 빈 줄을 삽입하면서 자동 줄바꿈 적용
< pre > 
 pre 태그는 
  독  특  한 서식의 텍스트/코드를 나타내기에
          적합함
< /pre >
