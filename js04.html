알림창 출력

alert("메시지");
:"" 또는 '' 내에 메시지를 입력해야함.

- 웹 브라우저에서 알림창을 열어 메시지를 표시
- 가장 많이 사용하는 간단한 대화상자

< button type="button" id="alert"> < /button >

< script > document.quertySelector(#alert).onclick = function(){
    alert("안녕하세요!");
}
</script>

확인창 출력

confirm("메시지");
:큰 따옴표 또는 작은 따옴표를 사용

- 버튼이 2개인 창
- [확인] [취소] 버튼 중 하나를 클릭할 수 있음
- 클릭 결과를 변수에 저장하고 그 값에 따라 프로그램 동작

< button type="button" id="confirm"> < /button >

< script > document.quertySelector(#confirm).onclick = function(){
    confirm("확인 또는 취소를 눌러 창을 닫아주세요.");
}
</script>

프롬프트 창에서 입력 받기

prompt("메시지");
prompt("메시지",값);
: (기본)값을 지정할 경우, 텍스트 안에 기본값이 표시됨

- 텍스트 입력 부분이 있는 작은 창
- 텍스트 창에 간단한 메시지 입력 가능 >> html태그 사용 불가
- 입력한 내용을 받아, 프로그램에 사용 가능

< button type="button" id="alert"> < /button > < script > document.querySelector(#prompt).onclick = function(){
    prompt("가고 싶은 여행지를 입력해주세요.","시드니");
}
</script>
