속성 선택자

: 특정 속성을 가진 요소를 선택해 스타일을 적용

속성 선택자 사용
태그명[ 속성 ] { 스타일 규칙 }
>>태그명은 생략시 [속성]을 가지는 모든 요소에 적용됨.

h2 [ title ]{ background-color:ccffff; }

속성 선택자 적용
< h2   title="test" >

title속성을 가진 h2태그

< /h2 >
< h2   title="test2" >

title값이 다른 < h2 >에도 적용됨

< /h2 >
< p   title="test3" >

h2의 title 선택자이므로, title을 가진 p태그에는 적용 안됨

< /p >

하위 선택자

Desendant Selector
: 특정 선택자 안에 포함된 하위 요소들을 선택하여 스타일 적용 가능

하위 선택자 사용
상위태그 하위태그 { 스타일 규칙; }
ㄴ> 공백으로 구분하여 나열

div p{
   color:purple;
   font-family:"고딕",gungseo,serif;
}

하위 선택자 적용
< div >
< p >

< div >의 하위 태그인 < p >에 스타일 적용됨

< /p >
< /div >

자식 선택자

Child Selector
:부모 요소 내에 (직속)포함된 자식 요소를 선택해 스타일을 적용
- 자식 요소: 상위 태그(부모 요소)의 바로 하위에 존재하는 내부 태그
ㄴ두 번째 하위에 존재하게 되는 태그(후손/하위/손자..)는 자식 요소가 아님
*하위 선택자와 헷갈리니 주의!

자식 선택자 사용
부모태그 > 자식태그 { 스타일 규칙; }
ㄴ > 로 구분
.parent > span {
   border:3px double blue;
   padding:10px;
   font-size:12px;
}

자식 선택자 적용
< div class="parent" >
< span >  <= .parent의 하위 태그
parent클래스인 div의 직속 자식인 p태그
< /span >< br >
   < div >
   < span >    .parent인 div태그의 하위의 하위 태그인 p태그    < /span >  <= .parent의 직속 자식이 아닌 손자!
   < /div >< br >

< /div >
