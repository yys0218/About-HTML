스타일의 상속

: 부모 요소(상위 태그)에 적용한 스타일이 자식 요소(하위 태그)에 적용되는 것
- 하위 요소에 상속되지 않는 속성도 있다.
>> 배경색, 배경이미지, border, margin, padding, float, positon은 상속되지 않는다.

상속을 활용한 크기 지정
em
:부모의 값을 기준으로 해, 상대적으로 변하는 단위
n em = 부모의 속성값의 n 배
- 상속이 가능하며, em으로 글자 크기를 지정한 경우, 상대적으로 작거나 크게 나올 수 있다.

div {
   border:1px solid red;
   padding: 2em;
}
p {
   font-size: 1.5em;
}

em의 적용
< div > <= *body의 padding의 2배
< p > <= *div의 폰트 사이즈의 1.5배

부모요소인 div의 폰트사이즈의 1.5배
padding값 2배



< /p >
< /div >

 * px과 em의 차이점
	- px 단위 적용: body 요소(부모 요소)의 크기를 무시하고 직접 지정한 px로 재설정
	- em 단위 적용: body 요소(부모 요소)의 크기를 기준으로 상대적으로 크기를 설정
