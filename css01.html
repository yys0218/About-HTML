CSS

:Cascading Style Sheets
- 웹 문서의 내용과 상관없이 디자인만 바꿀 수 있다.
- HTML: 웹 사이트의 내용 구성 후-> CSS: 웹 문서의 디자인 구성
>>내용과 디자인이 구분되어, 내용 또는 디자인을 수정할 때 서로 영향을 미치지 않는다.

-CSS 코드 한 줄이 하나의 스타일에 해당
선택자 { 속성: 속성값; 속성2: 속성값2; 속성3:속성값3;,...}

선택자: 문서의 레이아웃과 관련된 스타일 지정위해 선택되는 (태그)
- 자바스크립트에서 요소를 구별할 때에도 사용됨
{ }: 스타일 규칙이 들어감
스타일 규칙: 속성과 속성값을 짝지어 부르는 말
; : 스타일 규칙을 구분, 여러 스타일을 지정할 수 있게 함

스타일 시트(Style Sheet)
웹 문서 내 여러 개의 스타일 규칙 적용 시
>> head태그 내의,
< style > 안에 정의하기 < /style >

< style >

/*

위 아래 어떤 식으로든 ok
