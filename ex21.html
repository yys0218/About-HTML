< button >
:클릭할 수 있는 버튼을 정의
-텍스트나 이미지와 같은 콘텐츠를 삽입할 수 있음
-브라우저별로 < button >에 대한 기본 타입이 다를 수 있으므로, type 속성값을 명시하는 것이 좋다.

button 태그의 속성 - 이름과 값
name="이름" : button요소의 이름 - 태그의 데이터 참조시 변수명처럼 사용됨.
*같은 이름을 가진 여러개의 button 요소가 각각 다른값을 가질 수도 있음
value="값" : button의 초기값 - 버튼 선택 시 제출되는 데이터 값

button 태그의 속성 - type=~
="button" : 클릭할 수 있는 버튼
="submit" : 폼 데이터를 제출하는 버튼 - 입력이 완료됐으니, 지정된 액션을 실행한다는 의미를 가짐
="reset" : 폼 데이터를 초기값으로 리셋하는(입력사항을 지우는) 리셋 버튼
< button >이 속해있는 form 태그 안에서만 적용됨!

*< input type=button > 과 < button >의 차이*

 < select > , < option >
:옵션 메뉴를 제공하는 드롭다운 리스트를 정의
-select 태그는 option 태그를 요소로 함

< select >의 속성
name="이름" : 요소의 이름
size="한 번에 보일 옵션의 개수" : 옵션을 스크롤 형태로 보여주고, 그 때 보여질 < option > 의 개수를 지정
ㄴ미지정시 일반 선택창 형태(클릭시 전체 옵션이 보여짐)
< option >의 속성
- 아무 속성도 명시하지 않아도 되지만, 서버로 제출되는 값을 명시해주는 것이 일반적
value="값": 요소의 값

<form>통신사 선택
<select name="choice"> <option value="KT">KT</option>
<option value="U+">U+</option>
<option value="SKT">SKT</option>
<option value="알뜰폰">알뜰폰</option>
</select> <button type="submit">선택 완료</button>
<button type="reset">다시 선택</button>
</form>
