< font >

:글자의 표현 형태(스타일)를 나타냄
여러 가지 속성/속성값(attribute)을 가짐
명령 요소( < 태그명 > )- element
속성( 태그 내 설정 < 태그명 size="3" > )- attribute

< font >에서 사용할 수 있는 속성
: size, color, face,...
*< 태그 > 내에 작성되어야한다!

- size="숫자" : 폰트 크기를 지정하는 속성
ㄴsize는 1~7사이의 정수, 7이 가장 크고 1이 가장 작음
ㄴ속성은 시작 태그에만 적으면 됨
< font size="7" >가장 큰 폰트777< /font >
폰트666 폰트555 폰트444 폰트333 폰트222 폰트111
* 줄바꿈 자동 적용 안됨 >> 폰트 크기만 바뀜

- color="색이름 또는 색상 번호" : 폰트 색상을 지정하는 속성
ㄴ 색상명은 색 이름을 가리키는 영어 단어 red, blue,..
ㄴ 색상 번호는 RGB로 나타내는 값: rrggbb 에 해당하는 16진수(0~9, a~f)
*RGB색상표 검색하여 사용!
< font color="ff0000" > ffffff는 모든 색이 진함 >> 검정색 < /font >
< font color="purple" > 000000는 모든 색이 없음 >> 흰색 < /font >

- face=폰트 모양을 지정하는 속성
ㄴ글꼴을 지정 *글꼴표(font face)참조!
< font face="궁서" > 궁서체 < /font >
< font face="바탕체" > 바탕체 < /font >
< font face="Comic Sans MS" > Comic Sans MS < /font >
< font face="Impact" > Impact < /font >

속성의 조합: 한 번에 여러 속성을 적용하려면 띄어쓰기로 구분해 나열하면 됨
< font face="바탕체" size="2" color="fusia">
속성의 값은 "(큰 따옴표)로 묶어주는 것이 원칙이나,
값에 띄어쓰기가 포함되지 않는다면 생략도 가능
< /font >
< font size="-1" face="고딕" color="yellowgreen"> 속성의 순서는 상관 없음. 속성=값 만 잘 짝지어지면 됨. < /font >

< 시작 태그 >
작성해야할 내용이 많으면 종료태그를 줄바꿔 내린 다음
그 사이에 탭하여 내용을 적으면 됨.
어쨌든 내용이 태그로 감싸지기만 하면 됨.
< /종료 태그 >
