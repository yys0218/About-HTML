< textarea > : 여러 줄 텍스트 입력할 수 있는 입력창 생성

textarea 태그의 속성
name="이름" : 요소의 이름
rows="행 개수" : 한 번에 보여질 세로줄의 수 - 기본값은 2
cols="영역의 가로 너비" : 한 번에 보여질 가로 너비 - 기본값은 20 (2=약 한글자 정도)
maxlength="최대 문자수" : 입력 허용되는 최대 문자수

<form> <textarea name="text" rows"5" cols="20" placeholder="5줄씩 보여짐"></textarea> </form>

< label > : 라벨 태그의 요소 클릭 시 같은 이름의 id가 지정된 태그에 자동으로 포커스가 되도록(커서가 가도록)함
- 라벨 태그에 for="이름" 지정 후, 연동하고 싶은 태그에 id="이름" 지정하면 됨

label 태그의 속성
for="이름": label태그의 이름 지정
id="for=과 같은 이름": label과 연동하고 싶은 태그에 지정해야함

<form> <label for="msg">메시지 입력하기</label><br>
<textarea name="text" id="msg"></textarea><br> <input type="reset" value="다시 쓰기">&nbsp;<input type="submit" value="전송"> </form>
