css의 색상 적용

색상명
.keyword { color: red ; }
keyword클래스 선택자의 색상은 red

16진수 표현법
- #을 붙여 16진수 표현법이라는 걸 브라우저에게 알림
- rrggbb 순으로 각 자리에 0~f의 숫자를 입력

.hexa { color:#0000ff; }
hexa클래스 선택자의 색상은 #0000ff

10진수 표현법
- rgb(r의 정도, g의 정도, b의 정도)
- 각 자리에 0~255의 수

.decimal { color:rgb(255,200,30) }
decimal클래스 선택자의 색상은 rgb(255,200,30)

백분율
- rgb(r의 정도, g의 정도, b의 정도)
- 각 자리에 백분율

.percentage{ color:rgb(0%,100%,0%) }
percentage클래스 선택자의 색상은 rgb(0%,100%,0%) 


* 속성(스타일규칙) 지정시 : 과 =""의 차이

  - 기본적으로, 스타일 규칙은 속성:속성값;
  p { color:blue; }

  - 인라인 스타일로 태그 내 style의 값으로 속성 지정시 style="스타일 규칙"
  < p style="color:blue" >
  < font color="blue">
