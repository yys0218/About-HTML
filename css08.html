형제 선택자

: 같은 부모를 가진 요소들 간의 관계에서,
먼저 나오는 요소를 형 요소, 그 다음에 오는 요소를 동생요소 라고 함.

< 부모 요소 >
   < 자식 요소1 > <= 형 요소
   < 자식 요소2 > <= 동생 요소
< /부모 요소 >
< 부모 요소의 형제 요소 >

*모든 태그는 body태그 내에 작성되므로, 같은 들여쓰기 단계에 있는 태그는 형제라고 볼 수 있음!

인접 형제 선택자

: 형 요소에 인접한(바로 다음에 오는) 동생 요소에게 스타일을 적용

인접 형제 선택자 사용
형 요소 + 동생 요소 { 스타일 규칙 }
ㄴ> +로 구분 h3 + h4{
   color: blue;
   background-color: yellow;
}

인접 형제 선택자 적용
< h3 >

형 요소

< /h3 >
< h4 >

형 요소에 인접한 동생 요소

< /h4 >
< h4 >

형 요소에 인접하지않은 동생 요소

< /h4 > <= *같은 태그(h4)라도 형 요소 (h3)와 붙어 있지 않으면 스타일 적용X

일반 형제 선택자

: 형 요소 다음에 오는 모든 동생 요소에 스타일 적용

일반 형제 선택자 사용
형 요소 ~ 동생 요소 { 스타일 규칙 }
ㄴ> ~로 구분 h4 ~ h5{
   color: yellow;
   background-color: blue;
}

일반 형제 선택자 적용
< h4 >

형 요소

< /h4 >
< h5 >

인접한 동생 요소
< /h5 >
< h5 >

인접하지 않은 동생 요소
< /h5 > <= *인접 형제 선택자와는 달리 모든 동생 요소에 스타일 적용
