About HTML

전체 선택자

Universal Selector
:문서의 모든 요소에 스타일을 적용함
- 태그에 들어있지 않은 것들에는 적용되지 않음

전체 선택자는 *(별)을 사용
* { margin-left:50px; }

타입 선택자

Type Seletor
: 특정 태그를 사용하는 모든 요소*에 스타일을 적용
- 모든 요소(태그)를 선택자로 사용할 수 있다.
- 인라인 스타일로는 사용 불가!

h3 { font-style:italic; color:#5678ff; }

h3태그를 사용하는 모든 요소에 적용됨

h3요소를 사용하는 모든 태그에 적용됨

< h3 > 인 모든 요소에 적용됨

    *
    태그 = 요소(element)
    태그(요소)가 가지는 속성(attribute)
