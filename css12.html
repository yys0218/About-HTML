가상 클래스
: 문서의 구조나 요소의 특정 상태를 기반으로 요소를 선택
- :로 가상 클래스(요소의 상태)를 표현함

자주 사용되는 가상 클래스

태그:hover { 스타일 규칙} : 요소 위에 마우스를 올렸을 때
.text:hover { color:purple; }
< span class="text" >마우스를 올리면 보라색이 됩니다.< /span >

태그:active { 스타일 규칙} : 요소를 활성화 했을 때(클릭한 순간)
button:active {
    background-color:navy; color:white;
}
< button type="button">

< /button >

태그:focus { 스타일 규칙} : 요소에 포커스가 있을 때(필드/링크의 활성화 등)
.input-box:focus {
    outline:none;
    border:2px solid skyblue; }
< form >< input class="input-box" type="text" placeholder="클릭시 테두리 변경">< /form >

a:link { color:blue; } : 방문하지 않은 하이퍼링크
a:visited { : 이미 방문한 하이퍼링크
    color:white;
    background-color:orange;
}
< a href="https://yys0218.github.io/About-HTML/css12.html" >현재 페이지 방문< /a >

first-child 가상 클래스

: 부모 요소가 가진 요소 중 첫 번째 자식요소에만 스타일을 적용
부모요소 자식요소:first-child { 스타일 규칙; }
.parent p:first-child { text-indent:2em; } *텍스트 들여쓰기
< div class="parent" >

< p >첫 번째 자식 요소< /p >

< p >두 번째 자식 요소< /p >

< p >세 번째 자식 요소< /p >

< /div >
